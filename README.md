
# Impostor Bíblico

Aplicación web tipo juego de roles para jóvenes cristianos, inspirada en el juego "Impostor", desarrollada con React 18 y Vite. No requiere backend: todo el estado se guarda en el navegador usando localStorage.

## ¿Cómo jugar?

1. Agrega los nombres de los jugadores (mínimo 3).
2. Inicia la partida: se asignarán roles al azar (uno será el Impostor, los demás Cristianos).
3. Cada jugador puede ver su rol en la pantalla.
4. ¡Jueguen y descubran quién es el Impostor!

## Tecnologías
- React 18
- Vite
- localStorage (sin backend)

## Instrucciones de desarrollo

1. Instala dependencias:
	```bash
	npm install
	```
2. Inicia el servidor de desarrollo:
	```bash
	npm run dev
	```
3. Abre [http://localhost:5173](http://localhost:5173) en tu navegador.

## Personalización
Puedes modificar los roles, reglas o interfaz editando los archivos en la carpeta `src/`.
